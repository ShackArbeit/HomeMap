Google Map API 簡介

Google Maps API 是一套由 Google 提供的應用程式介面（API），允許開發者在其應用程式中集成 Google 地圖的功能 !
本 Demo 將使用Google Maps API 所提供的 JavaScript API 、Places API 以及 Directions API 做功能展示 !
